## How to submit

1. Fork [avelino/awesome-go](https://github.com/avelino/awesome-go) on GitHub.
2. Clone your fork locally:
   ```
   git clone https://github.com/<your-username>/awesome-go.git
   cd awesome-go
   ```
3. Open `README.md` and find the **Machine Learning** section.
4. Add the entry above in alphabetical order by project name (infergo falls between any `i` entries and `j` entries).
5. Commit the change:
   ```
   git add README.md
   git commit -m "Add infergo — Go-native LLM inference server"
   git push origin main
   ```
6. Open a pull request from your fork to `avelino/awesome-go:main`.
7. In the PR description, confirm:
   - The project has a clear README with a quickstart
   - The project is Apache 2.0 licensed
   - It has been published to pkg.go.dev
   - It solves a problem not already covered in the list
